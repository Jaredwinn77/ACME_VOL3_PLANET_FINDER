Here is a link to a NASA dataset which classifies Kepler observations.  It contains the Kepler ID which arget identification number, as listed in the Kepler Input Catalog (KIC). The KIC was derived from a ground-based imaging survey of the Kepler field conducted prior to launch. The survey's purpose was to identify stars for the Kepler exoplanet survey by magnitude and color. The Kepler ID is unique to a target and there is only one Kepler ID per target.
https://www.kaggle.com/datasets/nasa/kepler-exoplanet-search-results
